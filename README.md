# Pricing Table Section - DevChallenges Solution

This is my solution to the **Pricing Table** challenge from the **DevChallenges.io** platform. 
The project focuses on building a modern, visually appealing 4-column pricing layout,
designed to master advanced CSS positioning techniques and responsive web design.

## 🎯 Project Goals
- Practice styling pricing cards and multi-column layouts using CSS.
- Master advanced layout positioning concepts such as **Relative (`position: relative`)** and **Absolute (`position: absolute`)** to place highlighted badges or structural elements.
- Enhance UI/UX design skills by creating a clean, modern, and user-friendly pricing interface.
- Improve understanding of responsive design to ensure the 4-column table transitions smoothly into a single-column stack on mobile viewports.

## 📋 Features & Requirements Fulfilled
According to the challenge specifications, the web page includes:
- **4-Column Layout:** Four distinct pricing tier options presented side-by-side using CSS Flexbox or Grid.
- **Detailed Tiers:** Each column includes key promotional elements: price points, featured options/services list, and an interactive call-to-action (CTA) button.
- **Advanced Positioning:** Implemented precise layout details using relative and absolute positioning techniques.
- **Responsive Design:** Fully optimized media queries that seamlessly adjust the card arrangements for optimal viewing on desktop, tablet, and mobile screens.

## 🛠️ Tech Stack
- **HTML5** (Semantic elements for pricing structure and accessibility)
- **CSS3** (Flexbox/Grid structure, custom relative/absolute positioning, and media queries)
- **Editor:** Visual Studio Code
